# Role Pack: Product Manager

## Recommended Integrations

| Integration | Why it matters for you |
|---|---|
| qmd hybrid search | Semantic recall across hundreds of pages — find the customer insight you heard 6 months ago |
| Obsidian sync | Live graph view of competitive relationships, customer clusters, and strategy threads |
| Gmail MCP | Ingest customer email chains directly into raw/internal/ without copy-paste |
| Slack MCP | Pull Slack threads into wiki as internal sources with thread provenance |

## Suggested Agent Mode

`team` — PMs benefit from worker delegation. Use worker-source-fetcher for batch URL ingests, worker-wiki-indexer after large ingests, worker-lint for quarterly health checks.

## Notes

- **Weekly rhythm**: ingest 3-5 analyst links → batch entity updates → refresh overview.md
- **Pre-meeting**: query customer entity → check persona page → surface open questions
- **Decision logging**: when you say "we decided" the agent should offer §5 Decision Journaling automatically
- Focus tags mean the agent surfaces competitive and customer pages proactively, even in passing mentions
